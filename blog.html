Conceptos básicos

PHP

Estructura

Cada instruccion debe terminar con ; y el codigo se escrib e entre <?php - ?>

<?php
  echo 'Hola';
?>

Variables

Las variables son debilmente tipadas. Se declaran con $ y pueden empezar con letra o _

<?php
 $var1 = 1;
 $name = 'Nombre';
?>

Podemos ver informacion de una variable con:

<?php
 $var1 = 1;
 var_dump($var1);
?>

Al usar comillas dobles podemos incluir variables dentro y php las va a remplazar en el caso de que existan, con comillas simples no sucede

<?php
 $lastName = 'Araujo'
 $name = "Francisco $lastName"
 echo ($name);
?>

Tipos de Variables

  • boolean: Representa solamente un valor verdadero o falso. 
    $a = true; 
$b = false;
  • integer: Representa un número entero positivo, negativo o 0. 
    $a = -123;
$b = 0;
$c = 7763;
  • float o double: Representa un número de punto flotante, existen problemas de precisión con los números flotantes debido a la naturaleza binaria de las computadoras. 
    $a = 12.24;
$b = 1.5e3;
$c = 7E-10;
  • string: Representa una cadena de caracteres.Existen 4 formas de representar una cadena. Las 2 principales son usando comillas simples o comillas dobles. 
    $a = ”Hola”;
$b = ‘Mundo’;

Variables compuestas:

  • array: Representa una colección de valores, aunque por defecto PHP usara índices numéricos, la realidad es que la estructura se representa como un mapa que colecciona pares llave-valor.
  • object: Representa una instancia de una clase. Este tema lo veremos más a fondo en la clase de Programación Orientada a Objetos.
  • callable: Es un tipo de dato especial que representa a algo que puede ser “llamado”, por ejemplo una función o un método.
  • iterable: A partir de PHP 7.1 iterable es un pseudo tipo de datos que puede ser recorrido.

Tipos especiales:

  • resource: Es un tipo de dato especial que representa un recurso externo, por ejemplo un archivo externo a tu aplicación.
  • NULLEs un valor especial que se usa para representar una variable sin valor.

Arrays

Declaramos un array agregando items separados por , dentro [ ] tambien podemos dar los indices manualmente de la forma: 1 => 'item 1' las llaves pueden ser enteros cadenas o arreglos

$var2 = [
 'valor1',
 'valor2',
 'valor3'
];

$var3 = [
 0 =>'valor1',
 1 =>'valor2',
 'key' =>'valor3'
];
